HTML-таблицы и СSS-inline стили

На втором занятии, я изучил:

  1. Теорию о том, что такое таблицы-HTML.
  2. Стили inline CSS

Примеры:

  • table,tr,td,tbody,th-теги для создания таблиц
  • style=".." - задать свойство\значение
  • height/width - высота, ширина
  • font-size - размер шрифта
  • text-align(left/right/center/justify) - расположения текста
  • font-weight(normal/bold) - жирность текста
  • font-style(normal/italic) - курсив
  • background-color(red/# RGB: #00ffff or #00ff ) -цвет фона
  • border:1 px solid #ccc- размер краев таблицы и цвет
