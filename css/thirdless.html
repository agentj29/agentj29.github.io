CSS Selector

На третьем занятии, я изучил:

  1. Где писать css
  2. Селекторы и их комбинации
  3. Как оформлять текст
  4. Как оформлять фон

Примеры:

  • основные селекторы - ID и Class
  • Class определяется как ".", ID как "#", их заносим в отдельный файл под названием "(text).css"
  • "*{..}"задать значение всем тегам.
  • Текст можно оформить такими атрибутами, как: font-size,font-style,font-weight, font-family,text-align,text-shadow.
  • Для установки изображение на фон используем атрибут "background-image: url(,,,.jpg), background-size"
